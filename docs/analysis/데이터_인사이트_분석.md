# 📊 보유 데이터 기반 이해관계자별 인사이트 분석

> **작성일**: 2026-03-04
> **데이터 범위**: 수도권(서울·인천·경기) 아파트

---

## 보유 데이터 요약

| 테이블 | 주요 컬럼 | 특징 |
|---|---|---|
| **apt_basic** | apt_id, apt_name, build_year, road_address, latitude/longitude, admin_dong | 단지 기본정보 + 위치 |
| **apt_detail** | household_count, approval_date, total_parking_count, subway_line/station, 편의/교육시설 | 시설·교통 상세 |
| **apt_trade** | deal_date, deal_amount, exclusive_area, floor, buyer/seller_type, deal_diff_rate | 매매 실거래 + 변동률 |
| **apt_rent** | deal_date, deposit, monthly_rent, rental_adjusted_deposit, deposit_diff_rate | 전월세 실거래 + 환산보증금 |
| **naver_complex** | complex_no, sgg_name, dong_name, latitude/longitude | 네이버 단지 위치 |
| **naver_listing** | article_no, trade_type(매매/전세/월세), initial_price, current_price, is_active, first/last_seen_date | 현재 매물 호가 + 이력 |

> [!TIP]
> 핵심 차별화 포인트: **과거 실거래가(apt_trade/rent)** + **현재 매물 호가(naver_listing)** 를 결합하면 시세와 호가 간의 괴리, 시장 온도 등 독자적 인사이트 도출 가능

## 1. 🏠 실수요자 (내 집 마련 / 이사 예정자)

실수요자는 **"지금 이 가격이 적정한가?"**, **"어디가 가성비가 좋은가?"** 를 가장 알고 싶어합니다.

### 인사이트 예시

#### 1-1. 호가 vs 실거래가 괴리 분석
> **질문**: "이 아파트 매물 8억인데, 실제로는 얼마에 거래되고 있나?"

| 분석 항목 | 사용 데이터 | 방법 |
|---|---|---|
| 현재 호가 | `naver_listing.current_price` | 활성 매물(is_active=true)의 중위 호가 |
| 최근 실거래가 | `apt_trade.deal_amount` | 동일 단지·면적의 최근 3개월 실거래 중위값 |
| **괴리율** | 두 테이블 조합 | `(호가 - 실거래가) / 실거래가 × 100` |

- **괴리율이 높으면** → 매도자 희망가가 높아 네고 여지 큼
- **괴리율이 낮거나 음수면** → 급매 가능성, 시세 하락 구간

#### 1-2. 동일 면적·동네 내 가성비 단지 비교
> **질문**: "강남구 34평대에서 가장 가성비 좋은 단지는?"

```
apt_basic (위치·연식) + apt_detail (세대수·주차·지하철) + apt_trade (평균 실거래가)
→ 평당가 기준 정렬 + 시설 점수 가산
```

- **출력 예시**: 평당가 낮은 순 + 지하철 도보 10분 이내 + 주차 세대당 1대 이상 필터

#### 1-3. 전세가율로 매수 적기 판단
> **질문**: "지금 전세가율이 높으면 매수해도 될까?"

| 지표 | 산출 | 의미 |
|---|---|---|
| 전세가율 | `rental_adjusted_deposit / deal_amount × 100` | 60% 이상이면 실수요 중심, 급락 리스크 낮음 |
| 매매-전세 갭 | `deal_amount - rental_adjusted_deposit` | 갭이 작을수록 투자 매력↑, 급락 시 역전세 리스크↑ |

#### 1-4. 매물 체류 기간 분석
> **질문**: "이 매물이 오래 안 팔렸다는 건 무슨 의미?"

- `naver_listing.last_seen_date - first_seen_date` = **매물 체류일**
- 체류일이 긴 매물 → 호가 고평가 또는 거래 침체 구간
- 단지 평균 체류일 vs 지역 평균 체류일 비교

## 2. 📈 투자자 (갭투자, 시세차익, 임대수익)

투자자는 **"어디가 오를 것인가?"**, **"수익률이 얼마인가?"** 에 집중합니다.

### 인사이트 예시

#### 2-1. 가격 변동 추세 및 모멘텀
> **질문**: "최근 3개월간 가장 많이 오른 지역은?"

- `apt_trade.deal_diff_rate` 활용 → 단지별·지역별 평균 변동률 집계
- 월별 추이 시계열 분석 (36개월 보존 데이터 활용)
- **행정동 단위 히트맵**: `apt_basic.admin_dong` + `apt_trade` 집계

#### 2-2. 호가 변동 추적 (네이버 매물)
> **질문**: "매물 호가가 올라가는 추세인가, 내려가는 추세인가?"

| 지표 | 산출 | 의미 |
|---|---|---|
| 호가 변동률 | `(current_price - initial_price) / initial_price × 100` | 양수면 호가 상승, 음수면 호가 하락(급매 전환) |
| 호가 하락 매물 비율 | 지역 내 `current_price < initial_price` 매물 비중 | 비율 높으면 하락 시장 신호 |

#### 2-3. 갭투자 적격 물건 탐색
> **질문**: "갭이 3억 이하이면서 향후 상승 가능성이 있는 단지?"

```
매매가(apt_trade 최근) - 전세가(apt_rent 환산보증금)  → 갭
+ deal_diff_rate 추세 상승 중
+ naver_listing 활성 매물 수  ← 적으면 희소성
```

#### 2-4. 월세 수익률 계산
> **질문**: "이 단지 월세 수익률은?"

- `apt_rent.monthly_rent × 12 / apt_trade.deal_amount × 100` = **연 수익률**
- 단지·면적별 수익률 랭킹
- 동별 평균 월세 수익률 맵

#### 2-5. 매물 회전율 (거래 활발도)
> **질문**: "이 동네 거래가 활발한가?"

- 활성 매물 수(`naver_listing` is_active=true) vs 월간 거래 건수(`apt_trade` 해당 월)
- 회전율 = 거래 건수 / 활성 매물 수 → 높을수록 시장 활발

## 3. 🏢 공인중개사 (시장 현황 파악, 고객 상담)

중개사는 **실시간 재고(매물) 현황**과 **거래 트렌드**로 고객 상담 역량을 높일 수 있습니다.

### 인사이트 예시

#### 3-1. 매물 재고 현황 대시보드
> **질문**: "우리 동네 매물이 얼마나 쌓여있나?"

| 지표 | 데이터 | 산출 |
|---|---|---|
| 거래유형별 매물 수 | `naver_listing.trade_type` | A1(매매), B1(전세), B2(월세) 각 활성 건수 |
| 신규 매물 증가율 | `first_seen_date` 기준 일별 신규 등록 추이 | |
| 매물 소진율 | `is_active=false` 전환 비율 | 높으면 거래 활발 |

#### 3-2. 적정 호가 제안
> **질문**: "고객에게 얼마에 내놓으라고 해야 하나?"

- 동일 단지·면적의 최근 실거래가 + 현재 활성 매물 호가 범위
- `deal_amount` 대비 `current_price` 분포 → 적정 호가 구간 제시

#### 3-3. 경쟁 매물 분석
> **질문**: "같은 조건의 매물이 몇 개나 있나?"

- 동일 단지 + 동일 면적대 + 동일 거래유형 → 경쟁 매물 수
- 가격대별 분포, 층수·향 비교 → 차별화 포인트 도출

## 4. 🏛️ 정책 입안자 / 연구기관

정책 결정자는 **시장 과열·침체 판단**, **지역 간 양극화**, **임차인 보호** 관점의 거시 데이터가 필요합니다.

### 인사이트 예시

#### 4-1. 지역별 시장 온도 지수
> **질문**: "어느 지역이 과열이고, 어느 지역이 침체인가?"

| 구성 요소 | 데이터 | 가중치 예시 |
|---|---|---|
| 거래 변동률 | `apt_trade.deal_diff_rate` 평균 | 30% |
| 호가 변동률 | `naver_listing` 호가 상승/하락 비율 | 20% |
| 매물 체류일 | `last_seen - first_seen` 평균 | 20% |
| 전세가율 | 환산보증금 / 매매가 | 15% |
| 거래 회전율 | 거래 건수 / 활성 매물 수 | 15% |

→ 종합 지수를 행정동·시군구 단위로 산출하여 **시장 온도 히트맵** 생성

#### 4-2. 매매/전세 양극화 분석
> **질문**: "수도권 내 가격 격차가 심해지고 있나?"

- 시군구별 평당가 상위 10% vs 하위 10% 격차 추이 (36개월)
- 전세가율의 지역별 편차 → 임차인 부담 수준 비교

#### 4-3. 매수자/매도자 유형 분석
> **질문**: "법인 매수가 늘고 있나?"

- `apt_trade.buyer_type`, `seller_type` 집계
- 법인 vs 개인, 외지인 비율 등 → 투기 과열 여부 판단

#### 4-4. 전월세 전환율 모니터링
> **질문**: "전세의 월세 전환이 가속화되고 있나?"

- `naver_listing`에서 B1(전세) vs B2(월세) 매물 비율 추이
- `apt_rent`에서 전세 vs 반전세 vs 월세 거래 비율 시계열

## 5. 🏦 금융기관 (담보대출, 리스크 관리)

금융기관은 **담보 가치 평가의 정확성**과 **하방 리스크 관리**에 관심이 있습니다.

### 인사이트 예시

#### 5-1. 실시간 담보 가치 평가
> **질문**: "이 아파트의 현재 시가를 어떻게 산정하나?"

- 최근 3개월 실거래 중위값 (면적·층 보정)
- 현재 활성 매물 호가 중위값
- 두 값의 가중 평균 → 보수적 담보 가치 산출

#### 5-2. 가격 하락 리스크 탐지
> **질문**: "하락 위험이 높은 지역은?"

| 위험 신호 | 지표 |
|---|---|
| 역전세 위험 | 전세가율 80% 이상 + 매매 하락 추세 |
| 매물 적체 | 활성 매물 ÷ 월 거래 건수 > 6 (6개월 이상 소진 예상) |
| 호가 하락 확산 | `current_price < initial_price` 비율 > 30% |
| 거래 절벽 | 전월 대비 거래 건수 50% 이상 감소 |

#### 5-3. 포트폴리오 리스크 분산도
> **질문**: "대출 포트폴리오가 특정 지역에 편중되어 있나?"

- 담보 아파트의 `admin_dong` 기준 지역 분포
- 특정 행정동 집중도 → 리스크 분산 필요 여부

## 데이터 조합별 핵심 분석 매트릭스

| 조합 | 분석 가능 인사이트 |
|---|---|
| `apt_trade` + `apt_rent` | 전세가율, 갭, 월세 수익률, 매매-전세 추이 비교 |
| `apt_trade` + `naver_listing` | 호가-실거래 괴리율, 시장 온도, 매물 회전율 |
| `naver_listing` 단독 | 매물 재고, 체류일, 호가 변동, 신규/종료 추이 |
| `apt_basic` + `apt_detail` | 가성비 랭킹, 교통 접근성, 연식별 분석 |
| `apt_trade.buyer/seller_type` | 매수·매도 주체 분석, 법인 비율, 외지인 유입 |
| **모든 테이블 종합** | 행정동 단위 시장 온도 히트맵 (지도 시각화) |
